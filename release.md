# 版本发布和下载

BabaSSL使用Github的Release功能来实现版本的发布，请移步以下页面进行下载：

[https://github.com/BabaSSL/BabaSSL/releases](https://github.com/BabaSSL/BabaSSL/releases)

需要注意的是，BabaSSL目前只提供源代码的版本发布，暂不提供任何二进制版本的发布（包括但不限于RPM, DEB等）。
